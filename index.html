Марсоход Perseverance взял первый образец грунта для возврата на Землю

Робот размером с автомобиль, смог взять свой первый в истории образец с красной планеты, вырвав просверленный кусок из скалы, получившей название «Рошетт».

Это была вторая такая попытка Perseverance, которая также предприняла попытку отбора проб 5 августа.

Но порода, пробуренная в тот день, оказалась на удивление мягкой и рассыпалась на кусочки, которые не попали в предназначенную для этого трубу.

Windows 11 выйдет 5 октября

Компания Microsoft объявила дату выхода новой ОС Windows 11 — это 5 октября 2021 года.

С этого дня станет доступно бесплатное обновление до Windows 11 для соответствующих требованиям ПК с Windows 10 и станут доступны для приобретения новые компьютеры с Windows 11.
